# Changelog

## 0.2.0

- Add "Send to Trae" workflow that copies the optimized prompt and tries to open a compatible chat panel.
- Add optional OpenAI-compatible AI optimization.
- Store AI API key in VS Code/Trae SecretStorage.
- Add extension settings for API base URL, model, system prompt, and chat command ID.

## 0.1.0

- Initial Trae-compatible VS Code extension.
- Adds prompt optimizer webview.
- Adds optimize-selection command.
- Supports copy, insert, markdown export, and `.trae/rules` save flow.
